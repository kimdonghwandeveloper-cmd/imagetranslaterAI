# 🌏 Image Translater AI (Visual Translation Automation)

> **"외국어 포스터를 한국어 포스터처럼 감쪽같이!"**

이 프로젝트는 이미지 속에 있는 외국어를 자동으로 인식하고, 깔끔하게 지운 뒤, 자연스러운 한국어로 번역해서 다시 그려주는 **"자동 번역 AI 도구"**입니다. 복잡한 그래픽 툴을 몰라도 누구나 쉽게 현지화(Localization)된 이미지를 만들 수 있습니다.

---

## 🧐 이게 뭔가요?

혹시 해외 직구 상세페이지나 외국어 안내문을 보면서 *"이게 무슨 뜻이지?"* 하고 답답했던 적 있나요?
이 프로그램은 다음 4단계 작업을 **버튼 하나로** 자동으로 처리해 줍니다.

1.  **눈 (OCR):** "이게 무슨 글자지?" -> 이미지 속 글자를 찾아내서 읽습니다. (PaddleOCR)
2.  **지우개 (Inpainting):** "글자만 싹 지우자!" -> 배경 그림은 남기고 글자만 깔끔하게 지웁니다. (OpenCV / Stability AI)
3.  **뇌 (Translator):** "한국어로 바꾸자!" -> 문맥에 맞게 자연스럽고 매력적인 한국어로 번역합니다. (GPT-4o)
4.  **손 (Renderer):** "예쁘게 다시 쓰자!" -> 지워진 자리에 번역된 글자를 예쁜 폰트로 다시 그려 넣습니다.

---

## 🛠️ 준비물

이 프로그램을 실행하려면 딱 3가지가 필요합니다.

1.  **컴퓨터:** Windows, Mac, Linux 어디서든 됩니다. (Python이 설치되어 있어야 해요)
2.  **API 키:** 똑똑한 AI를 쓰기 위한 "출입증" 같은 것입니다.
    *   **OpenAI API Key**: 번역(GPT)을 위해 필요해요.
    *   **Stability AI API Key**: 배경 복원(지우개)을 위해 필요해요. (없어도 기본 기능은 작동합니다!)
3.  **Python & UV:** 프로그램을 돌리는 도구들입니다.

---

## 🚀 설치 및 실행 방법 (초보자용)

### 1단계: 설치하기

먼저 프로젝트를 다운로드 받고, 필요한 도구들을 설치합니다. 명령 프롬프트(터미널)를 열고 아래 명령어를 입력하세요.
(이미 `uv`가 설치되어 있다고 가정합니다. 없으면 `pip install uv`를 먼저 해주세요!)

```bash
# 1. 의존성 프로그램 자동 설치
uv sync
```

### 2단계: 설정하기 (API 키 넣기)

1.  폴더 안에 `.env.example` 이라는 파일이 보일 거예요. 이 파일의 이름을 `.env` 로 바꿔주세요.
2.  `.env` 파일을 메모장으로 열어서 여러분의 API 키를 붙여넣으세요.
    ```ini
    OPENAI_API_KEY=sk-어쩌구저쩌구...
    STABILITY_API_KEY=sk-어쩌구저쩌구...
    ```

### 3단계: 실행하기

이제 마법을 부릴 시간입니다!

```bash
# 프로그램 실행!
uv run verify_pipeline.py
```

실행하면 자동으로 샘플 이미지를 다운로드해서 번역을 진행합니다.
결과는 `assets` 폴더 안에 **`final_output.jpg`** 라는 이름으로 저장됩니다.

## 🎨 나만의 이미지 번역하기

내 컴퓨터에 있는 다른 이미지를 번역하고 싶다면?
`verify_pipeline.py` 파일을 메모장이나 코드 에디터로 열어서 아래 부분을 수정하면 됩니다.

```python
# 이미지 주소를 내 파일 경로로 바꾸세요
image_url = "C:/내/사진/폴더/poster.jpg" 
```

## ❓ 자주 묻는 질문

**Q. 글자가 이상하게 번역돼요.**
A. AI가 문맥을 모르면 그럴 수 있어요. `translator.py`에서 프롬프트(명령어)를 수정하면 더 좋은 번역 결과를 얻을 수 있답니다.

**Q. 배경이 깨끗하게 안 지워져요.**
A. 복잡한 무늬나 그라데이션은 지우기 어려울 수 있습니다. Stability AI API 키를 넣으면 훨씬 더 자연스럽게 지워줍니다.
